# Reference Examples

## Purpose

This file lists website references that help show the desired direction for the Viral Prints website.

These examples should guide contributors on:
- layout
- visual hierarchy
- portfolio presentation
- quote request flow
- trust-building
- customer conversion

## Desired Website Direction

The Viral Prints website should feel:
- bold
- modern
- visual
- premium
- creative
- easy to navigate
- strong on calls to action
