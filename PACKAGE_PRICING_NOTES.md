# Package Pricing Notes

## Purpose

This file outlines the recommended Phase 1 package structure for Viral Prints.

The goal is to make the offer easier to understand, price, and sell by creating standardized tiers.

---

## Package Tiers

### Core
Best for simpler, smaller projects with a streamlined scope.

Possible inclusions:
- standard size tier
- customer-supplied or simple catalog-based artwork
- standard proof round
- basic install assumptions

### Signature
Best for stronger branded visual impact and a more polished experience.

Possible inclusions:
- larger size tier
- stronger design support
- additional proof flexibility
- more customized visual direction

### Vision
Best for premium projects with greater visual presence or complexity.

Possible inclusions:
- largest size tier
- custom concept support
- higher-touch collaboration
- more advanced visual planning

---

## What the Package Structure Should Define

- what each package includes
- what is standardized
- what requires custom quote
- assumptions for revisions
- assumptions for install timing
- size tier guidance
- optional add-ons

---

## Suggested Add-On Categories

- custom design support
- expedited turnaround
- extra revisions
- special surface considerations
- after-hours install
- premium referral or analytics support later

---

## Pricing Philosophy

Phase 1 pricing should:
- be simple
- be easy to understand
- reduce decision fatigue
- make quoting faster
- still leave room for custom jobs where needed
